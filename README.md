<h1 align="center">Pixiviz</h1>

## 概述

一个具有清新画风，支持一定程度响应式的 Pixiv 代理站。

项目基于 Vue CLI + Element UI 编写。

## 开发目标

- [x] 完成Landing界面
- [x] 完成排行榜界面
- [x] 完成画作详情界面
- [x] 完成搜索界面
- [x] 榜单、搜索页面移动端适配
- [x] 查看大图窗口
- [x] 画师页面
- [x] Tag 跳转搜索
- [ ] 动图解析

## 使用方式

安装依赖：
```
npm install
```

运行本地服务器，实时编译 & 热重载：
```
npm run serve
```

推荐使用 Vue CLI 的 GUI 来管理本项目。
